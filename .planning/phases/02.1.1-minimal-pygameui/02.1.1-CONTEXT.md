<decisions>
## Implementation Decisions

### Bố cục bàn cờ (Board Layout)

- **D-01:** Bàn cờ vẽ dạng **hình thoi (diamond)** — lưới vuông xoay 45°.
- **D-02:** Vị trí 4 góc: ô 1 ở **đáy**, ô 9 ở **phải**, ô 17 ở **trên**, ô 24 ở **trái**. Di chuyển theo chiều kim đồng hồ 1→2→...→32→1.
- **D-03:** Mỗi ô là một hình thoi nhỏ (rotated square). Render bằng `pygame.draw.polygon`.
- **D-04:** Hiển thị nhãn mỗi ô: số thứ tự + loại ô (ký hiệu ngắn: `DAT`, `RUT`, `LE`, `TU`, `NGHI`, `THUE`, `DL`, `XP`, `MG`).
- **D-05:** Quân cờ (A/B/C/D) hiển thị bằng ký tự text (hoặc circle nhỏ) ngay trên ô, không dùng sprite. Nếu nhiều quân ở cùng ô → offset nhỏ để không chồng nhau.

### Panel thông tin player (Info Panel)

- **D-06:** Panel đặt **bên phải** cửa sổ, chiếm ~1/3 chiều rộng.
- **D-07:** Mỗi player hiển thị 3 dòng: **Tên player** / **Tiền hiện có (cash)** / **Tổng tài sản (cash + build cost đã đầu tư)**.
- **D-08:** 2–4 player xếp theo chiều dọc trong panel. Highlight player đang đến lượt.
- **D-09:** Phía dưới info panel là **event log** — hiển thị ~8–10 dòng sự kiện gần nhất (cuộn tự động, dòng mới thêm vào dưới cùng).

### Cách cập nhật UI

- **D-10:** UI **subscribe EventBus** — mỗi khi có `GameEvent` thì cập nhật state nội bộ của UI (không đọc trực tiếp từ Player/Board object trong render loop).
- **D-11:** GameController.step() chạy trong **background thread** riêng. Pygame render loop chạy ở main thread. Dùng `threading.Thread` + `threading.Lock` để bảo vệ shared state.
- **D-12:** UI giữ một bản copy nhẹ của game state (dict): `{player_id: {cash, total_assets, position}}` + `board_ownership: {position: owner_id}`. EventBus handler cập nhật dict này (thread-safe).

### Speed control

- **D-13:** 4 mức tốc độ: **Pause / 1x / 5x / Max**.
  - `Pause`: background thread dừng (sleep chờ resume signal).
  - `1x`: delay 800ms giữa mỗi `step()`.
  - `5x`: delay 160ms giữa mỗi `step()`.
  - `Max`: không delay, chạy hết tốc độ.
- **D-14:** Điều khiển tốc độ bằng **phím tắt**:
  - `Space`: toggle Pause / Resume (quay lại tốc độ trước đó).
  - `1`: 1x, `2`: 5x, `3`: Max.
- **D-15:** Hiển thị tốc độ hiện tại trong UI (ví dụ: `[1x]` hoặc `[PAUSED]`).

### Claude's Discretion

- Màu sắc ô đất: dùng màu đơn giản theo `color` group từ Board config (8 màu đã có trong `_COLOR_NAMES` của main.py). Resort màu khác biệt.
- Font: dùng pygame default font (không cần load file font riêng).
- Kích thước cửa sổ: 1280×720 (có thể điều chỉnh).
- Tổng tài sản tính theo công thức đã có: `cash + Σ(calc_invested_build_cost per owned tile)`.
- Entry point: `python main.py` (không có `--headless`) → mở Pygame window. Giữ `--headless` vẫn hoạt động như cũ.

</decisions>
