## Decisions Made
- Lazy import of GameView in `run_pygame()` isolates pygame: headless never touches it
- `_prev_speed` field tracks speed before Space-toggle pause so resume returns to correct speed
- Daemon thread: automatically dies with main process, no explicit cleanup needed
- `is_game_over()` checked both as while-loop guard and after `_pause_event.wait()` returns — handles race where game ends while paused
