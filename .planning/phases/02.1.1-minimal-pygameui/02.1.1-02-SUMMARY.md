## What Was Built

### BoardRenderer (`ctp/ui/board_renderer.py`)

- Diamond geometry with R=280, centre (427,360): corners at pos 1=(427,640), pos 9=(707,360), pos 17=(427,80), pos 25=(147,360)
- Precomputes all 32 tile centres in `__init__` using `_tile_center()` — static geometry, no recomputation per frame
- Draws rhombus tiles using `pygame.draw.polygon` with 4-point diamond shape (half-size s=24)
- 9 land group colours (groups 0-8) + 10 special space colours by SpaceId int
- 11 space abbreviations: LE, RUT, DAT, MG, TU, NGHI, XP, THUE, DL, MAY, TRUOT
- Ownership tint: 0.7x RGB multiplier when position in `board_ownership`
- Golden tile border: (255,215,0) 2px outline; normal: (30,30,30) 1px outline
- Tile labels: two lines (position number + abbreviation) centred at tile
- Player tokens: A/B/C/D letters with dark outline; per-spec offsets for 1-4 tokens per tile
- Bankrupt players excluded from token rendering

### InfoPanel (`ctp/ui/info_panel.py`)

- Panel region: x=853-1280, y=0-720 (427×720 px)
- Speed indicator row (28px): SPEED_LABELS from speed_controller; red=[PAUSED], green=[1x]/[5x]/[MAX]
- Player info blocks: divides INFO_H=512px equally per player count
  - Active player: blue highlight background + yellow name text
  - Bankrupt player: grey text + [PHA SAN] red label appended to name
  - 3 text lines: name (font 22), cash (font 18), total_assets (font 18)
- Event log (bottom 180px): up to 10 lines newest-last, "(no events yet)" fallback, long lines clipped via `screen.set_clip`
- All geometry constants exported: PANEL_X=853, PANEL_W=427, LOG_H=180, LOG_Y=540, SPEED_H=28
