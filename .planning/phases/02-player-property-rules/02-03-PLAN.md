# Plan 02-03: Gap Closure — MiniGame Log + Bankruptcy Sell Loop

## Objective

Fix 2 bugs phát hiện qua UAT Phase 2.

## Tasks

### Task 1: Fix MiniGame log key

**File:** `main.py`
**Line:** 250
**Fix:** `event.data.get('gain', 0)` → `event.data.get('result', 0)`

Root cause: log handler đọc key `'gain'` nhưng `GameStrategy` emit key `'result'` → nhan lai luôn in $0.

### Task 2: Fix Bankruptcy creditor path — dừng bán khi đủ tiền

**File:** `ctp/controller/bankruptcy.py`
**Fix:**
- Thay `while player.owned_properties:` (line 32) → `while player.cash < 0 and player.owned_properties:`
- Sau vòng while: chỉ chuyển tiền cho creditor và mark bankrupt nếu `player.cash < 0` (vẫn không đủ)
- Nếu `player.cash >= 0`: không mark bankrupt, không transfer tiền cho creditor

Root cause: creditor path bán TẤT CẢ property bất kể cash đã đủ hay chưa.
