Create `ctp/skills/pet_handlers.py`:

**Stamina wrapper pattern:** Each handler should check `player.pet_stamina > 0` before activating. On successful activation, `player.pet_stamina -= 1`. The engine's fire_pet() may already handle this — check engine.py implementation. If engine handles stamina, handlers just implement effect. If not, each handler must manage stamina.

**1. handle_pet_thien_than(player, ctx) -> list[GameEvent]:**
- Trigger: ON_CANT_AFFORD_TOLL (player cannot pay toll)
- Rate: tier-based (25/35/50/70/100%)
- Effect: Use angel card — skip toll payment. Set ctx["toll_override"] = 0
- Max stamina: 1 (fires at most once per game)
- Return PET_ACTIVATED event

**2. handle_pet_xi_cho(player, ctx) -> list[GameEvent]:**
- Trigger: ON_OPPONENT_COMPLETE_2_COLOR_PAIRS (opponent acquires/upgrades to have 2+ same-color CITY pairs)
- Rate: tier-based (25/35/50/70/100%)
- Effect: Build L1 on random empty CITY tile for player + that tile is protected from acquisition for 1 turn (set a flag or timer)
- Max stamina: 3
- Logic to detect "2 color pairs": check opponent's owned_properties, group by color (tile.color_group), count groups with >= 2 tiles
- Return PET_ACTIVATED event

**3. handle_pet_phu_thu(player, ctx) -> list[GameEvent]:**
- Trigger: ON_OPPONENT_ACQUIRE_YOURS (opponent acquires your property)
- Rate is the STEAL PERCENTAGE, not activation rate. PET_PHU_THU always activates but steals rate% of acquisition price. Tier rates: 50/75/100/150/200%.
- Effect: player.cash += steal_amount; ctx["opponent"].cash -= steal_amount
- steal_amount = acquisition_price * (tier_rate / 100)
- 200% means steal 2x the acquisition price
- Max stamina: 3
- Return PET_ACTIVATED event

**4. handle_pet_troi_chan(player, ctx) -> list[GameEvent]:**
- Trigger: ON_OPPONENT_PASS_YOURS (opponent passes your property during movement)
- Rate: tier-based (25/35/45/55/70%)
- Effect: Set ctx["opponent"].bound_turns = 1. Bound opponent must roll even to move; odd = skip move. This is checked at ROLL phase in FSM.
- Max stamina: 5
- Return PET_ACTIVATED event

Register all 4:

register_pet("PET_PHU_THU", handle_pet_phu_thu)
