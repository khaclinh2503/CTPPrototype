## Implementation Details

### SK_BUA_SET (ON_LAND_OPPONENT)

- D-51 enforced: returns None when `is_player_turn=False`
- Returns `{"type": "toll_waive", "move_to_nearest_unowned": True, "destination": int|None}`
- Uses forward wrap-around search (max 32 steps) to find nearest unowned CITY
- `destination=None` when all CITY tiles are owned — FSM skips repositioning

### SK_NGOI_SAO (ON_OPPONENT_LAND_YOURS, reactive)

- No turn restriction — fires in any turn (D-52)
- Effect 1: always returns `factor=2` when activated
- Effect 2: if opponent has `IT_CA_1` (Thiên Thần):
  - 70% (`randint(1,100) <= 70`): card destroyed, toll x2 applies
  - 30%: card succeeds, returns `toll_waived=True, angel_blocked=True`
- Angel card is consumed regardless of outcome (card is "used" in either case)

### SK_CUONG_CHE (ON_OPPONENT_MOVE_TO_YOURS, reactive)

- D-49 enforced: returns None when `is_opponent_turn=False`
- D-48 enforced: returns None when `opponent.position == target.position`
- Finds all L5 tiles in `player.owned_properties`, picks max by `calc_invested_build_cost`
- Fails silently (returns None) if no L5 properties
- Returns `{"type": "teleport", "destination": int, "movement_type": "teleport"}`

### Registry

All 3 handlers registered in `SKILL_HANDLERS` at module import time.
