## What Was Built

### Task 1: Foundation Data Layer

**Player state extensions (D-15):** 16 fields mới thêm vào Player dataclass:
- `rank`, `star`, `skills`, `pendants`, `pet`, `pet_stamina`, `pet_tier`
- `cam_co_decay_index`, `cam_co_current_rate`, `joker_pending`
- `skills_disabled_this_turn`, `cards_disabled_this_turn`, `consecutive_doubles`
- `bound_turns`, `dkxx_bonus_pool`, `pendant_rank`

**Board state extensions (D-13):**
- `illusion_position: int | None` — cho SK_AO_ANH
- `stop_sign_position: int | None` — cho SK_BIEN_CAM
- `acquisition_blocked_turns: int` trên Tile — cho PET_XI_CHO
- Cả 3 được reset trong `reset_for_new_game()`

**Schema redesign (D-10/D-33/D-39):**
- `RankConfig`: min_star, base_rate, chance — hỗ trợ D-05 formula
- `SkillEntry`: trigger (str hoặc list[str]), rank_config dict, always_active, secondary_rate
- `PendantEntry`: triggers list, rank_rates (B/A/S/R/SR), rank_rates_2 optional, always_active
- `PetEntry`: trigger, max_stamina, tier_rates list[float], steal_ratios optional

**YAML configs:**
- `skills.yaml`: 26 skills với exact rates từ GD files
- `pendants.yaml`: 12 pendants với B/A/S/R/SR rates
- `pets.yaml`: 4 pets với tier_rates và max_stamina

**ConfigLoader additions:**
- Properties: `board_config`, `skills_config`, `pendants_config`, `pets_config`
- `assign_random_loadout()` standalone function (D-16/D-34/D-41) với D-20 mutual exclusion SK_TEDDY/SK_GAY_NHU_Y

### Task 2: SkillEngine Core

**`ctp/skills/engine.py` — SkillEngine class:**
- `calc_rate(skill_cfg, player)`: D-05 formula, D-03 R->S fallback, D-04 return 0 if no config
- `fire(trigger, player, ctx)`: dispatch skills, D-51 disable check, T-02.5-02 max 10 handlers
- `fire_pendants(trigger, player, ctx)`: dispatch pendants bằng pendant_rank (D-31)
- `fire_pet(trigger, player, ctx)`: dispatch pet với D-40 stamina decrement, D-39 tier rate

**`ctp/skills/registry.py`:** SKILL/PENDANT/PET_HANDLERS dicts + `register_all(engine)` function

**`tests/test_skill_engine.py`:** 23 unit tests:
- TestCalcRate (6 tests): D-05, D-03, D-04
- TestFire (7 tests): dispatch, wrong trigger, disabled, rate=0, always_active, None return, DoS guard
- TestFirePet (5 tests): stamina decrement, stamina=0, wrong trigger, tier rate, no pet
- TestFirePendants (4 tests): pendant_rank lookup, wrong rank, disabled, wrong trigger
- TestRegistry (1 test): register_all()
