Create `ctp/skills/handlers_land_position.py`:

**SK_SUNG_VANG handler (ON_MOVE_TO_OPPONENT — when moving to tile with opponent):**
- D-51: Only fires during player's own turn
- ctx keys: `opponent`, `board`, `players`, `is_player_turn`
- **Effect 1 (always, 100%):** steal 15% of opponent's cash: `stolen = opponent.cash * 0.15; opponent.cash -= stolen; player.cash += stolen`
- **Interaction block:** During this interaction, opponent cannot active skills or use held cards
- **Effect 2 (rate-based):** teleport opponent to player's most expensive property (max calc_invested_build_cost). Movement type: Teleport (immune to traps). Tile effect triggers normally.
  - If player has no properties: Effect 2 skip
- Rate config: A: base=28, chance=2; S: base=38, chance=3

**SK_LOC_XOAY handler (ON_MOVE_TO_OPPONENT — when moving to tile with opponent):**
- D-51: Only fires during player's own turn
- ctx keys: `opponent`, `board`, `players`, `is_player_turn`
- Primary rate check, then secondary 60% fixed check
- On both pass:
  1. `opponent.skills_disabled_this_turn = True`
  2. `opponent.cards_disabled_this_turn = True`
  3. Teleport opponent to player's most expensive property (Teleport movement, immune to traps)
  4. Tile effect at destination triggers normally
  - If player has no properties: teleport skipped, disable still happens
- Rate config: A: base=38, chance=2; S: base=48, chance=3; secondary_rate=60

**SK_TOC_CHIEN handler (ON_LAND_TRAVEL — after Travel tile resolution):**
- ctx keys: `board`, `players`
- Effect: grant player an extra roll (return `{"type": "extra_roll"}`)
- Rate config: A: base=40, chance=2; S: base=50, chance=4

**Execution order when multiple skills fire at same tile:**
- SungVang E1 (steal) -> LocXoay check -> SungVang E2 (teleport) check
- Both fire independently

Register all 3 in SKILL_HANDLERS.
