**Shared helper (used by both SK_MC2 and SK_TRUM_DU_LICH E1):**
```python
def _grant_free_city(player, board, source_level):
    """Find random empty CITY tile, claim it at source_level for player."""
    unowned = [t for t in board.tiles if t.space_id == SpaceId.CITY and t.owner_id is None]
    if not unowned:
        return []
    chosen = random.choice(unowned)
    chosen.owner_id = player.player_id
    chosen.building_level = source_level
    return [GameEvent(EventType.SKILL_ACTIVATED, ...)]
```

**1. handle_sk_mc2(player, ctx) -> list[GameEvent]:**
- Trigger: ON_ACQUIRE (buy empty OR acquire from opponent)
- Call _grant_free_city() with level from ctx["tile"].building_level
- Per SK_MC2.md: identical logic to SK_TRUM_DU_LICH E1

**2. handle_sk_trum_du_lich(player, ctx) -> list[GameEvent]:**
- Multi-trigger skill with 2 effects:
  - E1 (ON_ACQUIRE): Call _grant_free_city() — same as SK_MC2
  - E2 (ON_LAND_RESORT): When landing at opponent's Resort, force acquisition. Check ctx["tile"].owner_id != player.player_id AND tile is RESORT. If active: transfer ownership (D-28: Resort 1 level, just change owner_id). Stub AI: always buy if can afford.
- Handler must check which trigger fired and branch accordingly

Register both:
