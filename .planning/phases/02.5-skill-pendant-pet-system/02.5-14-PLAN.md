**PT_CHONG_MUA_NHA handler (ON_OPPONENT_LAND_YOURS — at acquisition check time):**
- Rank rates: B:10, A:20, S:43, R:62, SR:70
- Effect: Block acquisition with compound rate
  ```python
  pendant_rate = getattr(cfg.rank_rates, player.pendant_rank, 0) / 100
  # active_factor by building_level: {1: 0.15, 2: 0.35, 3: 0.60, 4: 1.0, 5: 1.0}
  active_factors = {1: 0.15, 2: 0.35, 3: 0.60, 4: 1.0, 5: 1.0}
  level = tile.building_level
  active_factor = active_factors.get(level, 1.0) if level >= 4 else active_factors.get(level, 0)
  block_rate = active_factor * pendant_rate
  if random.randint(0, 99) < block_rate * 100:
      return {"type": "acquisition_blocked"}
  ```
- Example: SR (70%) at L3 (60%): block_rate = 0.70 * 0.60 = 42%
