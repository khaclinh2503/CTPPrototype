Create `ctp/skills/pendant_handlers_special.py`:

**1. handle_pt_dkxx2(player, ctx) -> list[GameEvent]:**
- Trigger: ON_DKXX_CHECK
- Effect: Boost DKXX activation rate by pendant rate (2/3/4/12/18%)
- Modify ctx["dkxx_bonus"] += pendant_rate_for_rank
- Return PENDANT_ACTIVATED event

**2. handle_pt_xich_ngoc(player, ctx) -> list[GameEvent]:**
- Dual trigger + dual rate:
  - R1 (ON_PRISON_ESCAPE_CHECK): Boost prison escape probability. Rate: 20/40/60/80/100%. Modify ctx["escape_bonus"] += rate1.
  - R2 (ON_DKXX_CHECK): Boost player.accuracy_rate by rate2 percentage points. Rate: 2/4/6/10/14%. This is a persistent boost for the remainder of the turn/check.
- Branch on ctx["trigger"]
- Return PENDANT_ACTIVATED event

**3. handle_pt_chong_mua_nha(player, ctx) -> list[GameEvent]:**
- Trigger: ON_OPPONENT_LAND_YOURS (when opponent lands on your property and wants to acquire)
- Effect: Block acquisition attempt. Formula from D-36:
  `block_rate = active_factor[building_level] * pendant_rate_for_rank`
  `active_factor = {1: 0.15, 2: 0.35, 3: 0.60, 4: 1.00}` (L1 through L4)
- If random check passes block_rate -> set ctx["acquisition_blocked"] = True
- Return PENDANT_ACTIVATED event

**4. handle_pt_sieu_sao_chep(player, ctx) -> list[GameEvent]:**
- Dual trigger + dual rate:
  - R1 (ON_OPPONENT_UPGRADE_SYMBOL): When opponent builds L5 (Bieu Tuong), player gets a random L5 on one of their owned properties. Rate: 5/7/10/15/30%. Pick random owned property with building_level < 5, set to 5.
  - R2 (ON_GAME_START): At game start, double starting cash. Rate: 10/15/20/75/90%. player.cash *= 2.
- Branch on ctx["trigger"]
- Return PENDANT_ACTIVATED event

Register all 4:
```python
register_pendant("PT_DKXX2", handle_pt_dkxx2)
register_pendant("PT_XICH_NGOC", handle_pt_xich_ngoc)
register_pendant("PT_CHONG_MUA_NHA", handle_pt_chong_mua_nha)
register_pendant("PT_SIEU_SAO_CHEP", handle_pt_sieu_sao_chep)
```
