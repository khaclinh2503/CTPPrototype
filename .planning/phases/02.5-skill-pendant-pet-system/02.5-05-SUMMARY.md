Position/same-tile + travel skills implemented: SK_SUNG_VANG steals 15% cash + rate-based teleport to most expensive property; SK_LOC_XOAY disables opponent skills/cards + secondary 60% teleport; SK_TOC_CHIEN grants extra roll after Travel tile.

## What Was Built

### SK_SUNG_VANG (Sừng Vàng)
- Trigger: `ON_MOVE_TO_OPPONENT`
- Effect 1 (100%): steals exactly 15% of opponent cash (`opponent.cash * 0.15`)
- Blocks opponent `skills_disabled_this_turn` + `cards_disabled_this_turn` immediately
- Effect 2 (rate-based): teleports opponent to player's most expensive property via `calc_invested_build_cost`
- Skips E2 if player has no owned properties
- D-51: returns `None` when `is_player_turn=False`
- Rate: A base=28 chance=2, S base=38 chance=3

### SK_LOC_XOAY (Lốc Xoáy)
- Trigger: `ON_MOVE_TO_OPPONENT`
- Primary rate check (engine's `calc_rate`) + secondary fixed 60% check
- On both pass: disables opponent skills + cards, then teleports to most expensive property
- Disable fires even when player has no properties (teleport skips silently)
- D-51 enforced
- Rate: A base=38 chance=2, S base=48 chance=3

### SK_TOC_CHIEN (Tốc Chiến)
- Trigger: `ON_LAND_TRAVEL`
- Returns `{"type": "extra_roll"}` — engine handles outer rate check
- Rate: A base=40 chance=2, S base=50 chance=4
