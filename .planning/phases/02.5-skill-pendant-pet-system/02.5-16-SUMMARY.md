## What Was Built

### ctp/skills/register_all.py

### ctp/controller/fsm.py — 52 skill_engine references, 20+ trigger points

**ROLL phase hooks:**
- Per-turn flag reset: `skills_disabled_this_turn=False`, `cam_co_decay_index=0`, `cam_co_current_rate` recalculated
- `joker_pending` resolved via `ON_JOKER_RESOLVE`
- `bound_turns` check: odd dice → skip MOVE, decrement counter
- `ON_DKXX_CHECK` pendant fire → `dkxx_bonus_pool` temp boosts accuracy_rate
- `ON_ROLL_BEFORE` skill fire
- `ON_ROLL_AFTER` skill fire → `_resolve_roll_modifiers()` applies dice_replace/dice_modifier/direction_choice

**MOVE phase hooks:**
- `ON_PASS_START` with D-47 additive stacking (`pass_bonus_pct` result type)
- `ON_MOVE_PASS` per tile during dice_walk (CamCo, PhaHuy)
- `ON_OPPONENT_PASS_YOURS` pet fire for each owner's tile passed

**RESOLVE_TILE phase hooks:**
- `ON_LAND_TRAVEL` + `ON_OPPONENT_TRAVEL`
- `ON_LAND_OWN` pendant fire (TuTruong, KetVang E2, BanTayVang)
- `ON_LAND_OPPONENT` + pendant → D-45 toll order: free-toll check first, then boost
- `ON_CANT_AFFORD_TOLL` pet fire (PET_THIEN_THAN)
- `ON_OPPONENT_LAND_YOURS` skill + pendant fire (reactive: property owner)
- `ON_MOVE_TO_OPPONENT`, `ON_OPPONENT_MOVE_TO_YOURS`, `ON_SAME_TILE`

**ACQUIRE phase hooks:**
- `acquisition_blocked_turns > 0` → skip acquisition entirely
- `ON_ACQUIRE` skill fire (MC2, TrumDuLich E1)
- `ON_OPPONENT_ACQUIRE_YOURS` pet fire (PET_PHU_THU)

**UPGRADE phase hooks:**
- `ON_UPGRADE` skill + pendant fire per upgraded tile
- `ON_UPGRADE_L5` cascade when new_level == 5 (max depth 3, T-02.5-17)
- `ON_OPPONENT_UPGRADE_SYMBOL` pendant fire for all other players
- `ON_OPPONENT_BUILD` pet fire for PET_XI_CHO owners

**END_TURN hooks:**
- `acquisition_blocked_turns` decremented on all tiles

**ENTER_PRISON hooks:**
- `ON_ENTER_PRISON` fire (SK_JOKER) on triple-doubles
- `ON_PRISON_ROLL` pendant fire before escape roll attempt

**GAME_START:**
- `fire_game_start()` public method fires `ON_GAME_START` for all players

### main.py
- Imports SkillEngine, register_all_handlers, assign_random_loadout
- Creates SkillEngine with skills/pendants/pets config
- Calls register_all_handlers(skill_engine)
- Calls assign_random_loadout() to distribute loadout to all players
- Passes skill_engine= to GameController
- Calls controller.fire_game_start() before game loop
