**SK_AO_ANH handler (ON_UPGRADE or ON_LAND when at own property/L5):**
- Triggers: `["ON_UPGRADE", "ON_LAND"]`
- ctx keys: `tile`, `board`
- Check ON_LAND: `tile.owner_id == player.player_id and tile.building_level == 5`
- Check ON_UPGRADE: always valid (any upgrade level)
- Effect: Place illusion at current tile position
  ```python
  board.illusion_position = tile.position
  board.illusion_owner_id = player.player_id  # creator immune (GD note)
  ```
- Only 1 illusion at a time — new placement overwrites old (any player's)
- Illusion does NOT affect the player who created it (checked in handlers_move.check_traps)
- When other player steps on/through illusion: stop, tile effect triggers, illusion consumed
- Rate config: A: base=50, chance=2; S: base=60, chance=3

**SK_BIEN_CAM handler (ON_UPGRADE to L5, or ON_LAND at own L5):**
- Triggers: `["ON_UPGRADE", "ON_LAND"]`
- ctx keys: `tile`, `board`, `new_level` (for ON_UPGRADE), `choose_fn`
- Check ON_UPGRADE: `new_level == 5` (only L5 triggers)
- Check ON_LAND: `tile.owner_id == player.player_id and tile.building_level == 5`
- Effect: Player chooses any tile position on map to place stop sign
  ```python
  chosen_pos = choose_fn(board)  # stub AI: choose opponent's most expensive property
  board.stop_sign_position = chosen_pos
  ```
- Only 1 stop sign at a time — new placement overwrites old
- Stop sign affects EVERYONE including creator (unlike illusion)
- Persists until triggered (no turn expiry)
- When any player steps on stop_sign: stop, tile effect triggers, sign consumed
- Rate config: A: base=45, chance=2; S: base=55, chance=3

NOTE: Board also needs `illusion_owner_id: str | None = None` field (add in board.py if not already present from Plan 01).

Register both in SKILL_HANDLERS.
