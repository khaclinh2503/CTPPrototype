## What Was Built

### ctp/skills/pendant_handlers_own.py

**PT_TU_TRUONG (`handle_tu_truong`)**
- R1 (always, any own property): Refund % of invested build cost via `calc_invested_build_cost()`. Rates B:4%, A:10%, S:15%, R:25%, SR:50%.
- R2 (rate check, only at L5): Pull opponents within 4-tile circular radius to player position. Rates B:4%, A:5%, S:10%, R:25%, SR:42%. D-48: skip opponents already at same tile.

**PT_BAN_TAY_VANG (`handle_ban_tay_vang`)**
- Only fires at building_level == 5 (D-24).
- Pulls all opponents on same board side via `board.get_row_non_corner_positions()` (D-27).
- D-48: skips opponents already at player position.
- Rates B:2%, A:5%, S:20%, R:50%, SR:85%.

**PT_TUI_BA_GANG (`handle_tui_ba_gang`)**
- E1 (ON_OPPONENT_LAND_YOURS, always_active): Returns `{"type": "toll_boost", "percent": N}`. Rates B:10%, A:20%, S:30%, R:50%, SR:60%.
- E2 (ON_SAME_TILE, always_active): Steals % of cash from all opponents at same tile position. Rates B:5%, A:7%, S:10%, R:15%, SR:36%.
- Uses `ctx["trigger"]` to dispatch between E1 and E2.

**PT_KET_VANG (`handle_ket_vang`)**
- E1 (ON_OPPONENT_LAND_YOURS, always_active): Returns `{"type": "toll_boost", "percent": N}`. Rates B:10%, A:20%, S:30%, R:50%, SR:60%.
- E2 (ON_LAND_OWN, always_active): Refunds % of invested build cost via `calc_invested_build_cost()`. Rates B:10%, A:15%, S:25%, R:50%, SR:60%.

**Toll boost stacking**: Both PT_TUI_BA_GANG E1 and PT_KET_VANG E1 return `toll_boost` results independently. FSM/caller sums them additively (e.g., SR rank: 60 + 60 = 120% total boost).
