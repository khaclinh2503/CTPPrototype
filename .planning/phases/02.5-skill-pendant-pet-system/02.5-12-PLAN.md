**1. handle_pt_giay_bay(player, ctx) -> list[GameEvent]:**
- Trigger: ON_LAND_TRAVEL
- D-36: Free visit (skip toll) + move to travel destination tile
- Rate: 1/3/5/10/15% per B/A/S/R/SR
- Effect: Set ctx["toll_override"] = 0 AND allow travel teleport
- Return PENDANT_ACTIVATED event

**2. handle_pt_cuop_nha(player, ctx) -> list[GameEvent]:**
- Trigger: ON_LAND_OPPONENT
- D-36: % chance to steal property AFTER paying toll
- Rate: 5/7/10/15/25% per B/A/S/R/SR
- Effect: If active, transfer tile ownership (tile.owner_id = player.player_id). Toll already paid before this check.
- Return PENDANT_ACTIVATED event

**3. handle_pt_mang_nhen(player, ctx) -> list[GameEvent]:**
- Trigger: ON_LAND_OPPONENT (dual-rate)
- D-36: R1 reduces toll by 100% (3/5/8/20/30%), R2 steals property (5/7/10/15/25%)
- Check R1 first: if active, ctx["toll_override"] = 0
- Check R2 independently: if active, transfer ownership
- Both can fire in same activation
- Return PENDANT_ACTIVATED events

**4. handle_pt_sieu_taxi(player, ctx) -> list[GameEvent]:**
- Dual trigger: R1 on ON_LAND_TRAVEL, R2 on ON_LAND_OPPONENT
- R1 (30/40/50/66/76%): Instant move to another travel tile after landing
- R2 (3/5/8/20/35%): Free visit (skip toll) at opponent property
- Branch on ctx["trigger"]
- Return PENDANT_ACTIVATED event
