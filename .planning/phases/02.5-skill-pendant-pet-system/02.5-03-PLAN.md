**Trap check helper:**
```python
def check_traps(board, player, tile_position, movement_type):
    """D-55: Check illusion/stop_sign during Dice Walk, Sweep Walk, or Travel Walk.
    Teleport and Skill Walk immune (skip entirely).
    Returns: (should_stop: bool, trap_type: str|None)
    """
    if movement_type in ("teleport", "skill_walk"):
        return False, None
    # Illusion check
    if board.illusion_position == tile_position and board.illusion_position is not None:
        # illusion does NOT affect the player who created it
        # (creator tracking needed — use board.illusion_owner_id)
        board.illusion_position = None  # consumed
        return True, "illusion"
    # Stop sign check
    if board.stop_sign_position == tile_position and board.stop_sign_position is not None:
        board.stop_sign_position = None  # consumed
        return True, "stop_sign"
    return False, None
```

NOTE: Board needs `illusion_owner_id: str | None = None` to track who placed the illusion (SK_AO_ANH does not affect its creator). Add this in the handler when setting illusion.
