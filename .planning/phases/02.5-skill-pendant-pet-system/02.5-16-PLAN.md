**Specific FSM injection points:**

ROLL phase (_do_roll or equivalent):
- Before dice roll: fire("ON_ROLL_BEFORE", ...) — check ctx["dice_override"]
- After dice roll: fire("ON_ROLL_AFTER", ...) — check ctx["move_override"], ctx["move_direction"]
- DKXX check: fire("ON_DKXX_CHECK", ...) — check ctx["dkxx_bonus"]
- Check joker_pending at ROLL start: if player.joker_pending, give reward, reset flag
- Check bound_turns: if player.bound_turns > 0, check dice parity. Odd = skip move, decrement bound_turns

MOVE phase (_do_move or equivalent):
- Per-step loop: fire("ON_MOVE_PASS", ...) for each tile passed
- Per-step: fire_for_opponents("ON_OPPONENT_PASS_YOURS", ...) for property owners of passed tiles
- Per-step: check board.illusion_position — if player steps on it AND not owner, stop movement, clear illusion
- Per-step: check board.stop_sign_position — same logic

RESOLVE_TILE phase:
- After landing: fire("ON_LAND", ...) for general land triggers
- If opponent tile: fire("ON_LAND_OPPONENT", ...) — check ctx["toll_override"]
- If travel tile: fire("ON_LAND_TRAVEL", ...)
- If resort tile: fire("ON_LAND_RESORT", ...)
- If same tile as opponent: fire("ON_MOVE_TO_OPPONENT", ...)
- Reactive for tile owner: fire_for_opponents("ON_OPPONENT_LAND_YOURS", ...)
- If can't afford toll: fire_pet("ON_CANT_AFFORD_TOLL", ...)

ACQUIRE phase:
- After acquisition: fire("ON_ACQUIRE", ...)
- For previous owner: fire_pet("ON_OPPONENT_ACQUIRE_YOURS", ...)
- After acquire/upgrade: check ON_OPPONENT_COMPLETE_2_COLOR_PAIRS for all opponents

UPGRADE phase:
- After upgrade: fire("ON_UPGRADE", ...) with ctx["new_level"]
- If L5: fire_for_opponents("ON_OPPONENT_UPGRADE_SYMBOL", ...)

PRISON:
- On enter prison: fire("ON_ENTER_PRISON", ...)
- On escape attempt: fire("ON_PRISON_ESCAPE_CHECK", ...) — check ctx["escape_bonus"]

PASS START:
- When passing start: fire("ON_PASS_START", ...) — check ctx["bonus_multiplier"]

END_TURN:
- Reset per-turn flags:
  ```python
  player.skills_disabled_this_turn = False
  player.cards_disabled_this_turn = False
  player.cam_co_decay_index = 0
  player.cam_co_current_rate = 0.0
  ```

GAME START (in GameController.start_game() or equivalent):
- fire("ON_GAME_START", ...) for each player

Reactive triggers (fire in opponent's context):
- ON_OPPONENT_MOVE_SKILL: after any skill moves a player, check all opponents
- ON_OPPONENT_TRAVEL: after player reaches Resort, check all opponents

CRITICAL: All fire() calls must be guarded with `if self.skill_engine:` to maintain backward compatibility with existing tests that create GameController without skill_engine.
