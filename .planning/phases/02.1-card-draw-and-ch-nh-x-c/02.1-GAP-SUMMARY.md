## Decisions Made

1. **EF_7 and EF_8 both produce toll=0**: At player-level, virus means no toll on ALL owned tiles. The tile-level distinction (0% vs 50%) no longer applies. Both set `virus_turns=3` per D-11.
2. **Tile-level toll_debuff_turns kept as-is**: Existing tile-level decrement loop preserved in `_do_end_turn` (harmless; tile fields still exist on Tile dataclass).
3. **Early clear on visitor land**: When visitor lands on any tile owned by virus-debuffed owner, `owner.virus_turns` resets to 0 immediately per D-22.
