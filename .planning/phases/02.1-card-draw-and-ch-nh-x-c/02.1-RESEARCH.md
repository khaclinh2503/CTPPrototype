## Risk Flags

### RISK-01: Card.json có IT_CA_13 ≠ CONTEXT.md D-26
**Vấn đề:** CONTEXT.md D-26 viết "IT_CA_13 Ra Đảo EF_13" nhưng Card.json thực tế IT_CA_13 = EF_11 (festival_tour). IT_CA_14 = EF_13 (go to prison).
**Resolution:** Implement theo Card.json thực tế. IT_CA_13 = EF_11, IT_CA_14 = EF_13. Planner cần map card ID → effect ID từ Card.json, không từ CONTEXT.md card name labels.

### RISK-02: FortuneStrategy cần card pool injection
**Vấn đề:** `TileRegistry.resolve(SpaceId.CHANCE)` trả về `FortuneStrategy()` không arg. Nếu thêm `__init__(self, card_pool)` thì TileRegistry pattern phải thay đổi.
**Resolution options:**
- Option A: Lazy load Card.json trong `on_land()` (module-level singleton cache). Không cần thay đổi registry.
- Option B: TileRegistry cho phép factory function thay vì class. Phức tạp hơn.
- **Recommend Option A:** `_CARD_POOL: dict | None = None` module variable, load một lần.

### RISK-03: virus_turns clear khi "bất kỳ player nào visit"
**Vấn đề (D-11):** "Clear về 0 khi bất kỳ player nào visit bất kỳ tile nào của player này." Điều này xảy ra trong `LandStrategy.on_land()` và `ResortStrategy.on_land()` khi check virus. Logic trong D-44 step 1 đã handle: `if owner.virus_turns > 0: owner.virus_turns = 0; return events (no toll)`. Nhưng cũng phải áp dụng khi **chủ đất tự ghé** (không trả toll) — cần check thêm nhánh `tile.owner_id == player.player_id`.
**Resolution:** Trong `LandStrategy.on_land()` và `ResortStrategy.on_land()`, thêm check virus clear cả khi player là chủ (elif is_own branch).

### RISK-04: Teleport card không kích hoạt Start bonus

### D-38: Precision roll algorithm
```python
def _resolve_can_luc(self, chosen_range: int) -> tuple[int,int]:
    lo, hi = RANGES[chosen_range]
    precision_check = random.randint(1, 100)
    if precision_check <= self.current_player.accuracy_rate:
        # Hit: random T trong khoảng, split thành (d1,d2)
        T = random.randint(lo, hi)
        d1 = random.randint(max(1, T-6), min(6, T-1))
        d2 = T - d1
        return (d1, d2)
    else:
        # Miss: normal 2d6
        return (random.randint(1,6), random.randint(1,6))
```

### D-39: AI target-aware range selection
```python
def _choose_range_ai(self, player: Player, board: Board, players: list[Player]) -> int:
    """Chọn khoảng lực tối ưu dựa trên vị trí đích."""
    # Tìm unowned CITY/RESORT gần nhất
    best_step = None
    for steps in range(1, 13):
        target_pos = ((player.position - 1 + steps) % 32) + 1
        tile = board.get_tile(target_pos)
        if tile.space_id in (SpaceId.CITY, SpaceId.RESORT) and tile.owner_id is None:
            best_step = steps
            break
    if best_step is None:
        return 1  # Fallback: khoảng 1 [5-7]
    # Match khoảng
    for i, (lo, hi) in enumerate(RANGES):
        if lo <= best_step <= hi:
            # Check không phải đất đối thủ
            target_pos = ((player.position - 1 + best_step) % 32) + 1
            tile = board.get_tile(target_pos)
            if tile.owner_id and tile.owner_id != player.player_id:
                continue  # Ưu tiên khoảng khác
            return i
    return 1  # Fallback
```
